**React Explorer** is a hands-on React learning project built with **Vite + React**.
It demonstrates core React concepts through interactive examples, where each concept is implemented as a separate component and rendered dynamically inside `App.jsx`.

---

## ðŸ“Œ Features & Modules

### 1. JSX Module

* Demonstrates JSX syntax, expressions, and variables.
* Examples include dynamic rendering of names, IDs, and text content.

### 2. Props Module

* Shows passing data from parent to child components.
* Includes usage of renamed props (`heading`, `content`, etc.).

### 3. State Module

* Interactive counter using `useState`.
* Illustrates real-time state updates and automatic UI re-rendering.

### 4. List Rendering Module

* Renders arrays with `.map()` and unique keys.
* Demonstrates dynamic lists with updated fruit names.

### 5. Form Handling Module

* Controlled components with state-managed inputs.
* Handles user input, form submission, and displays data instantly.

### 6. Lifecycle & useEffect Module

* Demonstrates component lifecycle: mounting, updating, and unmounting.
* Includes a timer using `useEffect` with proper cleanup.

### 7. Class Component Module

* Shows React class component structure with:

  * constructor
  * state management
  * `this.setState()`
  * event handling
